# Stein Street Solutions

Public company/umbrella website for Stein Street Solutions.

## Domains

- `steinstreetsolutions.com` is the primary domain.
- `www.steinstreetsolutions.com` should redirect to the primary domain or be configured as the canonical domain in Vercel.
- Rastfuchs remains a separate app/project at `rastfuchs.de`.
- CCCR remains canonical at `ccc-study.org`.

## Deployment

This is a static site intended for Vercel.

Plain static site: `index.html`, `styles.css`, `vercel.json`. No build step.
